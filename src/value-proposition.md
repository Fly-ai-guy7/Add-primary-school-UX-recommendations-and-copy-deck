# Core Value Proposition

FocusFlow acts as five roles simultaneously:

| Role | Function |
|---|---|
| **Chief of Staff** | Manages priorities and coordinates work across projects |
| **Project Manager** | Tracks tasks, deadlines, and project progress |
| **Accountability Partner** | Checks in, motivates, and holds the user to their commitments |
| **Memory System** | Stores and retrieves all context in natural language |
| **Focus Coach** | Reduces overwhelm and guides single-task execution |

> The platform translates thoughts into action.
