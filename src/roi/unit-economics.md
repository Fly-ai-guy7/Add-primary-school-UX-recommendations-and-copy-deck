# Unit Economics

| Metric | Target | Notes |
|---|---|---|

---

## Why LTV:CAC Is So Favourable

The ADHD productivity market has two unusual characteristics that create exceptional unit economics:

1. **Community acquisition** — A significant portion of early users come through ADHD community referrals. These users arrive pre-sold on the problem and convert at 3–5x the rate of cold traffic. CAC is close to zero for community-referred users.

2. **High switching costs** — Once the Memory Vault contains a user's professional context, FocusFlow becomes genuinely hard to leave. This suppresses churn below SaaS averages for users past the 6-month mark.

## Monthly Cost Per Active User (AI Layer)

| Component | Estimated Cost Per User/Month |
|---|---|
| AI inference (GPT-5 equivalent) | EGP 30–50 |
| Infrastructure (cloud, database) | EGP 8–15 |
| Support and maintenance | EGP 5–10 |
| **Total variable cost** | **EGP 43–75** |

At Pro tier (EGP 500/month), contribution margin after variable costs: **EGP 425–457/user/month.**

This improves as:
- Model costs decline (AI inference costs have dropped 10x every 18 months historically)
- Usage patterns are optimised (not every user is a heavy API caller every day)
- Infrastructure amortises over a larger user base

## Break-Even Analysis

**Fixed monthly costs (Year 1, lean operations):**
- Engineering (founder salary + part-time help): EGP 50,000
- Infrastructure base cost: EGP 10,000
- Legal, admin, tools: EGP 8,000
- **Total fixed: ~EGP 68,000/month**

**Contribution margin per paying user:** EGP ~440/month (after variable costs)

**Break-even paying users: 155**

At 10% free-to-paid conversion, break-even requires **1,550 registered users** — a modest and achievable milestone within the first few months of launch.
