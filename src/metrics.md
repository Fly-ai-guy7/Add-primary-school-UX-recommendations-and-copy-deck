# Success Metrics

## Retention

| Metric | Description |
|---|---|
| **30-Day Retention** | Percentage of users still active 30 days after signup |

## Engagement

| Metric | Description |
|---|---|
| **Daily Active Users (DAU)** | Unique users engaging with the product each day |
| **Weekly Active Users (WAU)** | Unique users engaging at least once per week |
| **Average Focus Session Duration** | Mean length of active Focus Mode sessions |
| **Brain Dump Usage Frequency** | Average Brain Dump sessions per user per week |

## Outcomes

| Metric | Description |
|---|---|
| **Task Completion Rate** | Percentage of tasks created that are marked complete |
| **User Satisfaction Score** | NPS or CSAT collected at regular intervals |

---

## North Star Metric

**Task Completion Rate** — this is the one number that proves FocusFlow is fulfilling its core promise: translating thoughts into completed actions.
