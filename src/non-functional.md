# Non-Functional Requirements

## Performance

| Metric | Target |
|---|---|
| AI response time | < 3 seconds |
| Voice input processing | < 10 seconds |

## Security

- All data encrypted at rest and in transit
- Secure authentication (Google OAuth, Apple Sign-In)
- GDPR-compatible architecture
- Egyptian data residency options for enterprise deployments

## Availability

- High availability target for core features (Brain Dump, Focus Mode)
- Graceful degradation if AI layer is temporarily unavailable

## Accessibility

- Screen reader compatible
- High contrast mode
- Font size controls
- Reduced motion option for users with sensory sensitivities
