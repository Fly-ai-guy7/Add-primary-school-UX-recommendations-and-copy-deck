# Responsible AI Principles

FocusFlow is built on the following commitments:

## Transparency

The system explains its prioritisation decisions in plain language. Users always know why something has been surfaced as a priority.

## Non-Manipulation

FocusFlow does not use dark patterns, artificial urgency, or behavioural manipulation. Engagement is earned through genuine value, not psychological exploitation.

## User Autonomy

The user is always in control. AI recommendations are suggestions, not commands. Users can override, dismiss, or ignore any system output without penalty.

## Privacy

User data is never sold, shared, or used to train external models without explicit consent. The Memory Vault is private by design.

## Explainability

AI decisions are traceable. When the system makes a prioritisation choice, it can articulate the reasoning if asked.

---

> Aligned with responsible AI practices and the **Egyptian National AI Strategy** principles.
