# Module 2 — Focus Mode

## Purpose

Eliminate visual overwhelm by displaying exactly one task at a time, reducing the cognitive load that causes task paralysis.

## Interface Elements

| Element | Description |
|---|---|
| **Current task** | The single active task displayed prominently |
| **Next task** | A low-contrast preview of what comes after |
| **Estimated completion time** | AI-generated time estimate for the current task |
| **Start button** | Initiates the focus session |
| **Completion button** | Marks task done and advances to the next |

## Core Rule

> **Only one active task is ever visible.**

All other tasks are hidden. The user's only job is to work on the one thing in front of them.

## Design Rationale

Traditional to-do lists display 20–50 items simultaneously. For ADHD users, this creates decision paralysis and anxiety rather than clarity. Focus Mode inverts this — the list exists, but the interface surfaces only what matters right now.
