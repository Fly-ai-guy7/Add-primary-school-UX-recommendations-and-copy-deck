# Module 5 — AI Body Double

## Purpose

Simulate the cognitive support effect of "body doubling" — the well-documented ADHD phenomenon where having another person present (even silently) significantly improves focus and task initiation.

## Functions

| Function | Description |
|---|---|
| **Session initiation** | Starts a timed work session with a defined goal |
| **Check-ins** | Periodic prompts confirming the user is still on task |
| **Motivation** | Positive reinforcement and encouragement during sessions |
| **Task completion guidance** | Helps the user finish and close out the task cleanly |

## Example Interaction

> **FocusFlow:** "Let's work on this proposal for 15 minutes. I'll check in halfway through. Ready?"

> **User:** "Yes."

> **FocusFlow:** "Great. I'm here. Go."

*[8 minutes later]*

> **FocusFlow:** "Halfway there. How's it going?"

## Design Rationale

Body doubling works because the presence of another person activates the ADHD brain's social accountability circuits. FocusFlow's AI Body Double replicates this effect digitally — the user is never working alone, even when they are.
