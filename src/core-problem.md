# Core Problem

People with ADHD commonly experience:

| Challenge | Description |
|---|---|
| **Executive dysfunction** | Difficulty initiating, planning, and completing tasks |
| **Task paralysis** | Inability to start even when the path is clear |
| **Time blindness** | Impaired perception of time and deadlines |
| **Overwhelm** | Cognitive overload from too many competing demands |
| **Context switching** | Loss of state and momentum when interrupted |
| **Inconsistent follow-through** | Strong starts with weak finishes |
| **Information fragmentation** | Ideas and notes scattered across tools |

## The Root Issue

Current productivity tools require users to organise themselves **before** they can benefit from the tool.

This is a fundamental design flaw for ADHD users — the very skill these tools require is the skill ADHD impairs most.

**FocusFlow removes this requirement.**
