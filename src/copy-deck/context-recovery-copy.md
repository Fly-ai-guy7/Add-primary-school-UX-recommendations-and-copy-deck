# Copy Deck — Context Recovery

## Standard Return (30min–3 days away)

| Element | Copy |
|---|---|
| Headline | Welcome back. |
| Context label | You were working on: |
| Last action label | Last action: |
| Time remaining label | ~[N] min remaining |
| Primary CTA | Resume |
| Secondary CTA | Something changed |

---

## Return After 3+ Days

| Element | Copy |
|---|---|
| Headline | Welcome back. |
| Summary intro | Since you were last here: |
| New tasks added | [N] new tasks added automatically |
| Passed deadline | [N] deadline passed |
| Follow-ups | [N] items waiting on you |
| Primary CTA | See what needs attention |

---

## No Active Session on Return

| Element | Copy |
|---|---|
| Greeting — morning return | Good morning. |
| Greeting — afternoon return | Welcome back. |
| Greeting — evening return | Good evening. |
| First suggested action label | Where would you like to start? |
| CTA | See today's priorities |

---

## Edge Cases

| Scenario | Copy |
|---|---|
| App reopen, no prior session | *(No recovery screen — go to Today tab directly)* |
| Return mid-briefing window | Your morning briefing is ready. |
| Return during scheduled session | Your session was paused. Ready to resume? |
| Last task was completed before leaving | You finished everything on your list. |
