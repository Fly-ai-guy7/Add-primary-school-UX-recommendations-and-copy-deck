# Copy Deck — Momentum Mode

## Entry

| Element | Copy |
|---|---|
| Trigger button | Stuck? |
| Sheet headline | This one feeling stuck? |
| Option A | Yes |
| Option B | Not exactly — let me explain |

---

## Step 1: Acknowledge

| Scenario | Copy |
|---|---|
| User taps "Yes" | Okay. Let's make it smaller. |
| User taps "Not exactly" | Tell me what's going on. *(opens text input)* |

---

## Step 2: Reduce

| Element | Copy |
|---|---|
| Question | What's the smallest possible first action? |
| Input placeholder | e.g. Open the document |
| Confirm CTA | That works |
| Skip (no input) | Not sure — suggest something |

---

## AI Suggestion (When User Has No Micro-Step)

| Task Type | Suggested Copy |
|---|---|
| Writing task | Open the document and read the last paragraph. |
| Call / communication task | Find the contact and open a new message. |
| Research task | Open a browser tab and type the first question. |
| Administrative task | Open the file or form you need. |
| Unknown / generic | Find the file or tool you need and open it. |
