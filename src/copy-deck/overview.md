# Copy Deck — Overview

This copy deck defines all UI text for FocusFlow AI™ across every screen, state, and interaction. Copy is a first-class design material — for ADHD users, the words the product uses directly affect whether they feel understood, supported, or overwhelmed.

## Voice and Tone

### Voice Attributes

| Attribute | What It Means |
|---|---|
| **Direct** | Short sentences. No filler. Say the thing. |
| **Warm** | Supportive without being condescending. The tone of a trusted colleague, not a cheerleader. |
| **Confident** | Decisions are made on behalf of the user. No mealy-mouthed hedging. |
| **Non-judgemental** | Never implies the user should have done something already or is behind. |
| **Energising** | Subtly forward-leaning. Focused on what's possible, not what's missing. |

### Tone Shifts by Context

| Context | Tone Adjustment |
|---|---|
| **Morning briefing** | Calm, grounded, orienting |
| **Focus session** | Minimal, present-tense, action-focused |
| **Overwhelm state** | Softer, slower, reassuring |
| **Evening review** | Warm, reflective, forward-looking |
| **Error states** | Neutral, blame-free, solution-forward |
