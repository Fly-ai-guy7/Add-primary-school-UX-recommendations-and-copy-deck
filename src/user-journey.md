# User Journey

## Morning

```
1. Open FocusFlow
2. Receive Daily Briefing
   → Top 3 priorities
   → Calendar events
   → Pending follow-ups
3. Select first task
4. Enter Focus Mode
```

---

## During the Day

```
5. Capture thoughts using Brain Dump
   → Voice or text, any time
6. Receive AI prioritisation
   → System organises new input
7. Work with AI Body Double
   → Session support, check-ins, motivation
8. Use Momentum Mode if stuck
   → Smallest next step
9. Use Context Recovery after interruptions
   → Resume instantly
```

---

## Evening

```
10. Receive Evening Briefing
    → Completed items
    → Carry-forward tasks
    → Daily progress score
11. Generate tomorrow's priorities
    → AI proposes top 3 for next day
12. Store all context automatically
    → Memory Vault updated
    → Session closed
```

---

## Key Principle

The user never needs to maintain the system. FocusFlow maintains itself — capturing, organising, prioritising, and surfacing what matters at each moment of the day.
