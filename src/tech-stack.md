# Technology Stack

## Frontend

| Technology | Role |
|---|---|
| **Next.js** | React framework for server-rendered UI |
| **TypeScript** | Type-safe development |
| **Tailwind CSS** | Utility-first styling |

## Backend

| Technology | Role |
|---|---|
| **FastAPI** | High-performance Python API layer |

## Database

| Technology | Role |
|---|---|
| **PostgreSQL** | Primary relational data store |

## Authentication

| Provider | Method |
|---|---|
| **Google OAuth** | Social sign-in |
| **Apple Sign-In** | Social sign-in |

## AI Layer

| Stage | Model |
|---|---|
| **MVP (Primary)** | GPT-5 |
| **Future** | Claude, Gemini, DeepSeek |

---

> The AI layer is designed to be model-agnostic. The abstraction layer allows swapping or combining models without frontend or backend changes.
